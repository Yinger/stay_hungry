# Nginx 主要应用场景

- 静态资源服务
  - 通过本地文件系统提供服务
- 反向代理服务
  - Nginx 的强大性能
  - 缓存
  - 负载均衡
- API 服务
  - OpenResty

# Nginx 的组成

- Nginx 二进制可执行文件
- Nginx.conf 配置文件
- access.log 访问日志
- error.log 错误日志

# 参数配置

## 时间单位

- ms milliseconds
- s seconds
- m minutes
- h hours
- d days
- w weeks
- M months，30 days
- y years，365 days

## 空间单位

- bytes
- k/K kilobytes
- m/M megabytes
- g/G gigabytes

## http 配置的指令块

- http
- server
- upstream
- location

# Nginx 命令行

- 格式：nginx -s reload
- 帮助：-? -h
- 使用指定的配置文件： -c
- 指定配置指令：-g
- 指定运行目录：-p
- 发送信号：-s
  - 立刻停止服务：stop
  - 优雅的停止服务：quit
  - 重载配置文件：reload
  - 重新开始记录日志文件：reopen
- 测试配置文件是否有语法错误：-t -T
- 打印 nginx 的版本信息，编译信息等：-v -V
